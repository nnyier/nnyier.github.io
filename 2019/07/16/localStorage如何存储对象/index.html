localStorage如何存储对象

JSON 对象提供的 parse 和 stringify 将其他数据类型转化成字符串，再存储到 storage 中就可以了

localStorage 中的键值对总是以字符串的形式存储

需要注意, 和 js 对象相比, 键值对总是以字符串的形式存储意味着数值类型会自动转化为字符串类型
