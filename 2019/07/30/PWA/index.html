安装@vue/cli-plugin-pwa引发的Vue工程问题

前言

我们项目想要应用 WPA 的 Add to Home Screen 功能，使我们的 Web 应用可以添加在手机主屏幕上
大致情况如下：


WPA

PWA（Progressive web apps，渐进式 Web 应用）运用现代的 Web API 以及传统的渐进式增强策略来创建跨平台 Web 应用程序。这些应用无处不在、功能丰富，使其具有与原生应用相同的用户体验优势。

参考：MDN-WPA

Add to Home Screen

添加到主屏幕（Add to Home Screen 或简称 A2HS）是现代智能手机浏览器中的一项功能，允许开发人员轻松方便地在主屏幕上添加代表他们喜欢的网络应用程序（或网站）的快捷方式，以便他们随后可以通过单个访问它。

具体的安装使用过程，请参考:MDN-Add-to-Home-screen

具体碰到的问题

  • node 版本
  • npm 版本
  • Vue 版本
